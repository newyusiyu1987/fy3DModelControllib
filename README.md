# fy3DModelControllib
经过长达2个月的研究，具有自主知识产权的三维可视化软件的研发接近尾声。本软件命名为fy3DModelControllib，采用三维可视化工具VTK，在VTK基础上结合地质需要，设计了一整套集数据、显示、操作的软件库，提供了良好的操作接口，可以轻松实现三维普通视图、切片视图、镂空视图，轻松实现顶视图、侧视图、面视图。
